# BI-Dashboard-Development-DataLens-

## Описание проекта

В рамках проекта были разработаны аналитические дашборды в DataLens для анализа транзакционных данных.

Цель — визуализировать ключевые метрики, выявить закономерности в данных и упростить анализ пользовательской активности и продаж.

---

## Данные

В проекте использовались транзакционные данные, содержащие:

- информацию о покупках (сумма, скидка, себестоимость)  
- данные о клиентах  
- данные о доставке  
- дату и время покупки  
- адреса и реквизиты  

---

## Подготовка данных

Были созданы вычисляемые поля:

- **Почтовый индекс** — извлечение из адреса  
- **ИНН** — извлечение из строки покупателя  
- **Форма организации (АО)** — определение типа компании  
- **Город** — извлечение из адреса  
- **Дата (разбиение)**:
  - год, месяц, день, неделя, день недели  

- **Маржа**:
  (Сумма покупки - Сумма скидки) - Себестоимость
  
- **Тип получения**:
- Доставка  
- Самовывоз
